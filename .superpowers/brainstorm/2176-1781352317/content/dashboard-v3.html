Updated Rules — What Changed

✓ All fields optional

Name is the only required field. Phone and Email are fully optional — a guest can be added with just a name, then the host shares the invite link manually. No "at least one contact" gate.

Format validation on blur

Validation only fires if a value is entered and the field loses focus. Phone checked against /^\+?[\d\s\-()]{7,15}$/. Email checked against standard email regex. Empty field = no error, ever.

Submit blocked only on bad format

The Add button is disabled only if a field has an entered but invalid value. Empty phone + empty email = submit allowed. Invalid phone format = submit blocked until fixed or cleared.
